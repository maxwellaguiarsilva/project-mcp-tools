# Style Guides

This directory contains coding style guides and formatting conventions for the `project-mcp-tools` project.

---

## Available Style Guides

- **[Common Style Rules](all.md):** Language-agnostic rules shared by all guides (naming, comments, commas, single-use variables, language).
- **[Python Style Guide](python.md):** Defines the coding standards, naming conventions, and formatting rules for Python code in this project. Imports the common rules.
- **[C++ Style Guide](cpp.md):** Defines the formatting rules, naming conventions, and coding standards for C++ code in the host project (enforced via `cpp_code_verifier`). Imports the common rules.
