# SAK (Swiss Army Knife)

This directory contains the documentation for the `sak` (Swiss Army Knife) libraries across different languages. The `sak` libraries provide domain-agnostic utilities to elevate the level of development and promote domain-driven design.

## Available SAKs

- [C++ SAK](cpp.md)
- [Python SAK](python.md)
- [Lua SAK](lua.md)
- [Bash SAK](bash.md)
