# Python SAK

The Python `sak` library contains domain-agnostic utilities used primarily by the MCP tools and CLI scripts.

## Components

### `common.py`
Contains general utilities, such as:
- `ensure`: Runtime assertion utilities.
- `create_process`: Process execution wrappers.

### `fso/` (File System Operations)
A collection of utilities for interacting with the file system in a safe and predictable manner.
