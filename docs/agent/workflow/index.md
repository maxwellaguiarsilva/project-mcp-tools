# Workflows

This directory contains standard workflows and procedures for developers and AI agents working on this project.

---

## Available Workflows

- **[Simple Upload Workflow](upload.md):** Outlines the steps for quickly pushing code changes to GitHub without a pull request, including commit categorization and grouping.
- **[Consumer Submodule Synchronization](consumer-submodule-sync.md):** The workflow and safety nets for host projects consuming this framework as a git submodule.
- **[Markdown Reorganization Workflow](markdown-organization.md):** Instructs AI agents on how to audit, reorganize, and maintain markdown documentation to avoid redundancy and "markdown hell".
